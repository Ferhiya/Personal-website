Memory game

Use App

The game is a memory card matching game where the player's objective is to match all pairs of cards. At the beginning, a grid of facedown cards is presented. The player clicks on two cards to reveal their symbols. If the symbols match, the cards stay face up; otherwise, they are flipped back. The player repeats this process until all pairs are matched. Clicking the "Start Game" button initiates the timer and starts the game by activating the cards, which counts the time taken to complete the game. Once all pairs are successfully matched, a congratulations message appears.

Key features of the game include: Dynamic generation of memory cards with randomized symbols. Timer functionality to track the time taken to complete the game. Congratulations message displayed upon successfully matching all pairs, along with the time taken to complete the game.
